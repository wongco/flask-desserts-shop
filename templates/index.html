Welcome to the Ice Cream Shop!

All other endpoints in this app form a JSON API.

Here are all of the endpoints available to you:

Method Path Description
GET /desserts JSON data of all desserts
POST /desserts Adds a new dessert to our list (returns data on the new dessert)
GET /desserts/<id> JSON data on a single dessert
PATCH /desserts/<id> Update an existing dessert (returns data on the updated dessert)
DELETE /desserts/<id> Removes a dessert from our list (returns data on the deleted dessert)
